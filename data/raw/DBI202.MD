## Learning outcomes

| CLO Name | CLO Details | LO Details                                                                                                                                                      |
|----------|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1        | CLO1        | Understand thedatabase concepts and database management system software                                                                                         |
| 2        | CLO2        | Understand the relation model of data and Algebraic Query Language                                                                                              |
| 3        | CLO3        | Understand data normalization and apply normalization techniques in database design                                                                             |
| 4        | CLO4        | Be able to model an application’s data requirements using conceptual modeling tools like ER diagrams and design database schemas based on the conceptual model. |
| 5        | CLO5        | Be proficient in structure query language including Data Definition Language(DDL) and Data Manipulation Language(DML)                                           |
| 6        | CLO6        | Understand PL/SQL concepts and manipulate with View, Cursors, Stored Procedures, Functions, Database Triggers                                                   |
| 7        | CLO7        | Apply the Index in database design and query optimization                                                                                                       |
