## Learning outcomes

| CLO Name | CLO Details | LO Details                                                                                                                                       |
|----------|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| 1        | CLO1        | Understand the basics of computer vision, its related fields, and its applications                                                               |
| 2        | CLO2        | Understand the concepts of image formation: image representation, lighting, image acquisition through the camera, camera calibration techniques. |
| 3        | CLO3        | Basic knowledge of image processing on points, lines, image filtering techniques and corresponding algorithm implementation                      |
| 4        | CLO4        | Understand the techniques of feature extraction and matching, implement the techniques of edges, lines extraction                                |
| 5        | CLO5        | Understand the basics of segmentation, techniques such as active contours, split and merge, mean shift and mode finding                          |
| 6        | CLO6        | Discuss feature-based alignments like 2D and 3D feature-based alignment                                                                          |
| 7        | CLO7        | Discuss the principles and the implementation of Image stitching                                                                                 |
| 8        | CLO8        | Understand and implement the object detection techniques                                                                                         |
| 9        | CLO9        | Explain the principles of face recognition, object tracking                                                                                      |

Correct Code Execution (50%)  Code runs without errors and produces the expected results.  Example:  Homography matrix is computed correctly and image is warped properly.  Edge detection code outputs clear edges (e.g., using Canny or Sobel).  Object detection code detects the correct number of objects in a test image.  2. Conceptual Understanding (20%)  The learner demonstrates understanding of the underlying theory.  Examples:  Can explain what a homography is (a transformation mapping points between two planes).  Understands the principle of edge detection (detecting intensity changes in the image).  Explains why normalization is needed before feeding images into a neural network.  If AI provides the code, the learner can rephrase and explain the method in their own words.  3. AI-assisted Skill (20%)  The learner uses AI effectively as a tool rather than relying on it blindly.  Examples:  Writes a precise prompt such as “Generate Python code for computing homography using OpenCV with four pairs of matched points.”  After AI provides code, the learner notices an error (e.g., wrong function arguments in cv2.findHomography).  Adjusts AI code to run on their actual dataset (e.g., changing image paths, adapting for grayscale vs. color images, handling batch inputs).  Simplifies redundant AI-generated code for better readability.  4. Critical Thinking (10%)  The learner goes beyond execution and reflects on results.  Examples:  Explains why homography fails when points are nearly collinear.  Analyzes why edge detection produces too many false edges in noisy images.  Points out that AI-generated code works only for synthetic data but not for real-world images.  Suggests simple improvements such as applying Gaussian smoothing before edge detection, or using RANSAC to improve homography robustness. |
